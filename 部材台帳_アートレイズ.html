GitHub トークンを設定
部材データと写真を保存するには GitHub Personal Access Token が必要です。 初回のみブラウザに保存します（GitHubには送信されません）。ダッシュボード・案件台帳と共用されます。
原価台帳から取込
「原価管理」スプレッドシートの行をそのままコピーして貼り付けてください（タブ区切り／CSV可）。
列の順番：ID / 案件番号 / 現場名 / 日付 / 店舗名・業者名 / 品目・内容 / 金額 / 勘定科目 / 原価区分 …
材料費・消耗品費の行を部材として自動抽出します（交通費・飲食・既に取込済みの行は自動で除外）。
